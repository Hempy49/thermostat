# Thermostat.js

## User Stories
```
As a user,
So that I can begin to use my new thermostat,
I would like its default temperature to be 20 degrees.

As a user,
So that I can turn up the heat when it's cold outside,
I would like to be able to increase the temperature.

As a user,
So that I can turn down the heat when it's hot outside,
I would like to be able to decrease the temperature.

As a user,
So that my house doesn't get too cold,
I would like the thermostat's minimum temperature to be 10 degrees.

As a user,
So that I can avoid large heating bills,
I would like my thermostat to have a power saving mode.

As a user,
So that I can avoid large heating bills,
I would like the max temperature to be fixed at 25 degrees when power saving is on.

As a user,
So that I can get VERY WARM if I want to,
I would like the max temperature to be increased to 32 degrees when power saving is off.
```
