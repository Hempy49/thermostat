# Thermostat.js

## User Stories
```
As a user,
So that I can begin to use my new thermostat,
I would like its default temperature to be 20 degrees.

As a user,
So that I can turn up the heat when it's cold outside,
I would like to be able to increase the temperature.

As a user,
So that I can turn down the heat when it's hot outside,
I would like to be able to decrease the temperature.
```
